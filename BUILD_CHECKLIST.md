## 🔄 خطوات البناء والنشر

### البناء المحلي
```bash
# 1. تنظيف المشروع
flutter clean

# 2. تثبيت الـ Dependencies
flutter pub get

# 3. تحديث الـ Dependencies
flutter pub upgrade

# 4. فحص الأخطاء
flutter analyze

# 5. تشغيل التطبيق
flutter run
```

### بناء Android
```bash
# 1. بناء APK
flutter build apk --release

# 2. بناء App Bundle
flutter build appbundle --release

# 3. التوقيع على الـ APK
jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 \
  -keystore ~/key.jks \
  build/app/outputs/flutter-apk/app-release.apk \
  alias_name
```

### بناء iOS
```bash
# 1. بناء IPA
flutter build ios --release

# 2. فتح Xcode
open ios/Runner.xcworkspace

# 3. إنهاء البناء في Xcode
```

### بناء Web
```bash
# 1. بناء الويب
flutter build web --release

# 2. النشر على Firebase Hosting
firebase deploy
```
